# Project-Local Skills

This repository keeps paper-writing Codex skills under `.codex/skills` so they are scoped to this project instead of the global `$CODEX_HOME/skills` directory.

When the user asks for undergraduate thesis planning, thesis rewriting, school-format finalization, plagiarism/AIGC report based rewriting, or codebase-grounded paper content, use these project-local skills by reading their `SKILL.md` files first:

- `.codex/skills/academic-paper-strategist/SKILL.md`
- `.codex/skills/academic-paper-composer/SKILL.md`

Do not install these skills into the global Codex skills directory unless the user explicitly asks for a global installation.
